# **App Name**: SwiftPay Mobile

## Core Features:

- Start Shopping Button: A prominent button that initializes the shopping session.
- Camera Scanner: Integrate camera functionality via the html5-qrcode library for scanning product barcodes.
- Product Simulation: Utilize a mock JS database to simulate real product data upon barcode scanning.
- Cart Updates: Real-time updating cart with item name, quantity, and price upon successful scan.
- Total Calculation: Automatically calculate and display the total amount of all items in the cart.
- Checkout Simulation: A 'Proceed to Payment' button to trigger a 'Payment Complete' confirmation.
- Session Reset: Button to 'Start New Session,' resetting the cart and preparing for the next shopping experience.

## Style Guidelines:

- Primary color: Dark blue (#2E3192) to inspire trust and security in financial interactions.
- Background color: Light gray (#E9E9F1), creating a neutral backdrop to ensure that product and cart information is prioritized, and provides sufficient contrast with primary color.
- Accent color: Purple (#7026A9) for key interactive elements, contrasting clearly against both the light background and the dark primary color.
- Body and headline font: 'Inter' sans-serif font known for its readability.
- Use simple, line-based icons for product categories and cart actions.
- Clear and simple grid layout, with the camera view taking precedence until checkout.
- Use subtle animations for when a product is scanned successfully and when the cart updates.
